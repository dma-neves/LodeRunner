Instructions:

Player:

  • Collect all the coins from a level to get access to the next
  • To move the player use the arrow keys for movement and space for shooting
  • Shooting creates a hole destroying a breakable tile under/next to the player. These holes regenrate after some time
  • Increase the score by collecting coins and killing enemies

    • Enemy:

  • If an enemy touches the player the game is lost
  • Enemies can collect coins and drop them after some time
  • Enemies will get stuck in the holes made by the player and drop a coin if they have one
  • If a hole regenerates with an enemy inside, this enemy will die
  • If an enemy dies with a coin, the game is lost
